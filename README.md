# EduLense

EduLense is a cross-platform app focused on a UI optimized for left-handed use
(layout flow, dominant-hand ergonomics, control placement, and gesture comfort).

## Repo Structure
- apps/flutter        -> Flutter mobile app
- apps/react-native   -> React Native mobile app
- apps/electron       -> Electron desktop app
- apps/react-web      -> React web app
- docs/               -> Documentation

## Setup Instructions (Will be updated)
### Prerequisites
- Node.js
- Git
- Yarn

### Running Projects
**React Web**
1. cd apps/react-web
2. npm install
3. npm run dev

**Electron**
1. cd apps/electron
2. npm install
3. npm run start

**React Native**
1. cd apps/react-native
2. npm install
3. npm run ios OR npm run android

**Flutter**
1. cd apps/flutter
2. flutter pub get
3. flutter run
